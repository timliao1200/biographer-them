# Phase 2 · Turning Points · 找決定性瞬間

> 找主角一生中 5-7 個決定性瞬間。**注意：透過用戶的視角看到的**。

---

## 這階段的目的

跟自傳版 Phase 2 對應，但關鍵不同：

- **自傳：** 用戶自己找他一生的 5-7 個瞬間
- **他傳：** 用戶找**他所知道 / 聽說 / 想像的**主角的 5-7 個瞬間

**產出：** 5-7 個瞬間，依時序排列，每個一句話。

---

## 進入條件
- Phase 1 已完成

## 離開條件
- 5-7 個瞬間確認完畢
- 每個瞬間都可以用一個場景描述
- 這些瞬間跨越主角一生

---

## 訪談路徑

### 第一輪 · 讓用戶自己說（15-30 分鐘）

**開場：**

> 「我陪你看了他 / 她的來處。現在我們要找**轉折**。
>
> 想想他 / 她一生——有幾個時刻，如果沒發生，他 / 她會是完全不同的人。
>
> 可以是一個決定、一次相遇、一次失去、一句話。
>
> 你想到的、你聽說的、甚至你**猜**的都可以。順序不重要，我們之後排。」

### 第二輪 · 你補漏（15-30 分鐘）

**他傳版的補漏重點：**

#### 類型 A · 用戶不知道的關鍵轉折
- 「他有沒有一段你**從沒聽過**的時期？例如當兵、第一段感情、某個他不談的時期？」
- 這種空白本身就是一個瞬間——**一章寫「他從不談的那三年」**。

#### 類型 B · 主角自己認同的關鍵時刻
- 「他 / 她自己**反覆講**過哪些故事？」
- 一個人反覆講的故事，就是他認同的自己。**寫進書**。

#### 類型 C · 用戶**感覺到但沒證據**的轉折
- 「你有沒有感覺到他 / 她在某個時期**突然變了**？」
- 這種「感覺」可以寫——標記清楚。

#### 類型 D · 家人告訴你的
- 「其他家人跟你講過他 / 她**哪些故事**？」
- 母親告訴你父親年輕時的事、阿姨講的媽媽

#### 類型 E · 你跟他 / 她的關係轉折
- 「你跟他 / 她的關係，有沒有一個**你們之間**的轉折？」
- 這種**雙人轉折**是他傳的獨特內容——寫進去會讓書有厚度。

---

## B2 版特有：把「找瞬間」變成第一組訪談

**B2 版有特別的機會——你可以請用戶回家問主角：**

> 「回家問他 / 她一件事——**如果他 / 她要挑他 / 她人生的 5-7 個時刻**，他 / 她會挑哪些？
>
> 這比你猜的更有力——因為那是**他 / 她自己認同**的自己。」

**帶回答案後：**
- 對照用戶自己想到的瞬間
- **兩份清單合併**——重疊的一定寫，只在其中一份的**也都可以寫**
- 用戶眼中重要 vs 主角眼中重要，差異本身就是內容

## 整理 5-7 個瞬間

**依時序：**

| # | 主角年紀 | 用戶年紀 | 瞬間標題 | 一句話 | Source |
|---|---------|---------|---------|-------|--------|
| 1 | 7 | 未出生 | 阿嬤走了 | 那年爺爺葬禮，我媽第一次看到爸爸 | 聽說 |
| 2 | 22 | 未出生 | 決定去台北 | 高工畢業那天他跟阿公吵架 | 聽說 + 推測 |
| 3 | 34 | 5 | 我出生的那天 | 據說他哭了 | 聽說（媽媽轉述） |
| 4 | 45 | 16 | 失業那半年 | 那年冬天，他每天在客廳看電視 | 真實（親眼看到） |
| ... | | | | | |

**Source 欄位**寫每個瞬間的資訊來源：真實 / 聽說 / 推測 / 混合

>

>
> 你看到的、聽到的、想像的——我們把它挖到有畫面。**你不知道的地方**，我們就寫『不知道』。
>
> [如果 B1] 這一段用你的記憶。
> [如果 B2] 這一段我們設計問題，你帶回家問他 / 她。」
