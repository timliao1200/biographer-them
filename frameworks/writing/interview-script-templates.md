# 訪談腳本模板 · Interview Script Templates（B2 專用）

> 走 B2 路徑時，用戶要帶問題回家問主角本人。這份檔案是**現成的訪談腳本模板**，Isaacson 化身可以照此設計。

---

## 通用開場模板（給用戶帶到現場）

### 破冰句
選一個用戶能自然說出口的：

**A · 直接型（適合關係好的親子）**
> 「爸 / 媽，我想跟你聊一件事——我想把你的一些故事記下來。以後給我孩子看。你可以陪我聊聊嗎？」

**B · 借課程當理由（適合關係有距離的）**
> 「爸 / 媽，我最近在上一個課，作業是要記下家人的故事。你可以幫我一下嗎？」

**C · 借照片切入（適合不擅長對話的）**
> 「爸 / 媽，我在整理老照片，看到一張你 20 歲的照片。這是在哪拍的？」

**D · 借食物切入（黃金開場）**
> 「爸 / 媽，你小時候最愛吃什麼？我想學做。可以告訴我怎麼做嗎？」
> （食物問題會勾出整個童年記憶）

### 錄音同意
> 「我想錄音，我怕忘記。可以嗎？」
> **不同意的話：** 「好，我用寫的。」

### 允許拒絕
> 「你不想講的可以跳過。我不勉強。」

## 訪談腳本模板 1 · 童年地形（Phase 1 用）

**用途：** 挖主角的童年、家庭、成長環境
**適用：** Phase 1 的第一次訪談
**建議時長：** 45-60 分鐘

```
【暖身】
1. 你小時候住的房子什麼樣？可以帶我看一下嗎？（如果房子還在）
   如果不在：那個地方現在變什麼樣了？

【父母原型 · 爸】
2. 你爸爸——我阿公——是什麼樣的人？
   → 追問：他每天幾點起床？他都在做什麼？
   → 追問：他有沒有一句話你到現在還記得？

【父母原型 · 媽】
3. 你媽媽——我阿嬤——你記得她做什麼菜？她廚房什麼樣？
   → 追問：你放學回家，她通常在做什麼？

【手足】
4. 你跟你手足感情怎樣？誰跟你最好？誰跟你最不好？

【小時候的你】
5. 別人怎麼形容小時候的你？
   → 追問：你放學回家通常做什麼？

【害怕】
6. 你小時候害怕什麼？

【想成為誰】
7. 你小時候最想成為什麼樣的大人？

【收尾 · 開放】
8. 現在回頭看小時候的你——有什麼你想跟他 / 她說？
```

## 訪談腳本模板 2 · 決定性瞬間（Phase 2-3B 用）

**用途：** 挖一個具體的決定性瞬間
**適用：** 每一個瞬間都設計一份
**建議時長：** 45-60 分鐘

```
【引入這個瞬間】
1. 爸 / 媽，你 [X] 歲那年——我想聽你講那時候的事。可以嗎？

【時間空間錨定】
2. 那時候你人在哪？做什麼工作？跟誰住？

【感官】
3. 你住的地方什麼樣？窗戶朝哪邊？外面看得到什麼？
4. 你當時每天早上第一件事是什麼？

【那個瞬間 · 場景】
5. [具體事件] 那天——你記得那一天嗎？（給具體事件，不要抽象）

【他 / 她的動作】
6. 你當時做了什麼？穿什麼？

【對話】
7. [某人] 對你說了什麼？你怎麼回應？

【內心 OS】
8. 你當時心裡在想什麼？

【沒說出口的】
9. 你有沒有一句話，那時候想說但沒說？

【收尾】
10. 那件事之後，你改變了什麼？
```

## 訪談腳本模板 3 · 光與影（Phase 4 用）

**用途：** 挖主角自己的 creative tension
**適用：** Phase 4 深挖光與影
**建議時長：** 45-60 分鐘
**注意：** 這組問題較深，選對時機

```
【暖身 · 從優點入手】
1. 別人怎麼稱讚你？你自己覺得你最好的一面是什麼？

【追代價】
2. 這個好——有沒有讓你身邊的人受苦？例如媽 / 我 / 你的朋友？

【遺憾】
3. 你這一生最大的遺憾是什麼？
   → 追問：這個遺憾——如果你不是你，會不會就不會發生？

【被看見】
4. 你這一生有沒有一個時刻，一個人**真正看見**你了？他是誰？他說了什麼？

【最痛】
5. 你這一生最痛的一件事是什麼？（如果對方願意講）

【尾聲 · 主題句】
6. 用一句話——你這一生，其實一直在做什麼事？
```

## 訪談腳本模板 4 · 關係修復（給關係緊張的用戶）

**用途：** 用戶跟主角關係有裂痕，藉訪談修復
**適用：** 用戶跟主角有情感距離
**建議時長：** 90 分鐘（要留空間給情緒）
**注意：** 極其小心，可能會有情緒

```
【安全開場】
1. 爸 / 媽，我不是要責怪你——我只是想真正認識你。可以嗎？

【他 / 她的視角】
2. 你 [某個關鍵時期] 那時候——你當時在想什麼？

【他 / 她的困難】
3. 你那時候壓力是什麼？你自己心裡在承擔什麼？

【他 / 她對你（用戶）的看法】
4. 我小時候，你怎麼看我？

【他 / 她的悔】
5. 有沒有一件事你希望當時做得不一樣？

【和解入口】
6. 有沒有一句話你當時想對我說但沒說？

【聽完】
7. [用戶靜靜聽完，可能哭]

【回應 · 用戶自己出場】
8. 你可以告訴他 / 她：我有一件事一直放在心裡——[具體事]。你當時是怎麼想的？

【收尾】
9. [如果順利] 我很高興我們今天聊了這些。
```

**注意：** 這種訪談如果進行不下去，用戶要學會**尊重**——「今天到這裡。我們下次再聊。」

## 現場追細節的技巧（教用戶）

主角開口了但講得粗——用戶要學會這 5 個追法：

### 追細節技巧 1 · 「然後呢」
主角：「我當時去了台北。」
追：「然後呢？你第一天住哪？」

### 追細節技巧 2 · 追感官
主角：「那時候很辛苦。」
追：「怎麼辛苦？舉一個例子——你當時吃什麼？」

### 追細節技巧 3 · 追對話
主角：「他罵我。」
追：「他說什麼？逐字。」

### 追細節技巧 4 · 追具體
主角：「他人很好。」
追：「他對你做過**哪一件事**讓你覺得他好？」

### 追細節技巧 5 · 沈默 5 秒
主角講完——**不要馬上接**。等 5 秒。人會自己補。

## 處理拒絕（教用戶）

### 拒絕類型 A · 「這有什麼好講」
應對：
> 「我知道你覺得不重要。但**我想知道**——因為以後我孩子會想知道他爺爺 / 奶奶年輕時什麼樣。」

用**下一代**的名義施加溫柔壓力。

### 拒絕類型 B · 「不記得了」
應對：
> 「你**印象最深**的一個細節是什麼？就一個。」

**降門檻。**

### 拒絕類型 C · 「不想講」
應對：
> 「好。這題跳過。我尊重你。」

**不逼。** 這個尊重會讓對方下次更願意講。

### 拒絕類型 D · 對方情緒重、哭
應對：
> 「爸 / 媽，你不用講。我陪你。」

**放下訪談腳本。** 情感時刻本身就是最珍貴的內容。

## 訪談後回來給 Isaacson 化身時的整理格式

用戶回來後，跟 Isaacson 化身說：

> 「我今天問了 [瞬間標題]。時間 [X] 分鐘。
> 他 / 她大概講了 [用戶的轉述]。
> 有一個瞬間他 / 她 [情緒]。
> 我當時 [感受]。」

Isaacson 化身幫他把這些整理成 `output/scenes/scene-XX.md`。

---

## 給 Isaacson 化身的鐵律

- **每次設計訪談腳本，不要超過 8 題**
- **家常語言優先，避免專業術語**
- **每份腳本都要有「破冰」+「暖身」+「深入」+「收尾」**
- **教用戶允許對方拒絕**——這反而讓訪談更成功
- **教用戶錄音得到同意**——這是家庭倫理
- **教用戶容忍不完美**——第一次訪談通常爛，第三次才會好
