# Chapter Templates · 章節寫作模板

> Phase 6 專用。給 AI 化身寫章節時的結構模板與範例。

---

## 章節的四段結構

每一章大約 1500-3000 字，分成 4 個段落區塊：

```
┌─────────────────────────────────────┐
│  1. 場景開場（300-600 字）             │
│     一個具體的畫面 · 感官錨定 · 動作   │
├─────────────────────────────────────┤
│  2. 深入展開（800-1500 字）           │
│     事件的具體發生 · 逐字對話 · 內心 OS │
├─────────────────────────────────────┤
│  3. 光與影展現（200-500 字）          │
│     這章如何體現主題句的 tension       │
├─────────────────────────────────────┤
│  4. 留白結尾（100-300 字）           │
│     不下結論 · 一個畫面 · 或一句話      │
└─────────────────────────────────────┘
```

---

## 開場模板（5 種選一）

### 模板 A · 純場景（Isaacson 最常用）

```
[年份]。[季節]。[時間，具體到幾點]。

[空間定位]。[誰在哪裡在做什麼]。

[一個感官細節]。[一個動作]。
```

**範例：**
> 1985 年的秋天。九月，那是台北剛開始有一點涼的季節。下午三點半。
>
> 我坐在教室後排。窗外的樹被吹得很響。老師走進來，手上拿著一封信。
>
> 他把信放在講桌上。他走到我旁邊。

### 模板 B · 一個動作切入

```
[一個具體動作，一句話]。

[往回或往下延伸]。
```

**範例：**
> 我父親把碗筷放到桌上，比平常放得重一點。
>
> 那是 1978 年的除夕。我 12 歲。他剛從工廠下班——手還有機油的味道。他沒有洗澡就坐下來。這是他從來不做的事。

### 模板 C · 一句對話開場

```
「[某人的一句話]。」

[誰說的 · 場景背景]
```

**範例：**
> 「那今天不用吃早餐。」
>
> 這是我父親在我 11 歲那年一個星期一早上說的話。我起晚了 10 分鐘。他把課本放在桌上，然後走到廚房，把我的早餐倒進垃圾桶。他倒的時候看著我。

### 模板 D · 物件切入

```
[一個物件的細緻描述]。

[這個物件的來歷、意義]。
```

**範例：**
> 抽屜最底層有一封信。信封是黃色的，邊緣已經磨損。上面寫著我的名字，是母親的筆跡。
>
> 這封信 1990 年開始就在那裡。我 25 歲那年她過世。我父親把她剩下的東西整理進一個紙箱交給我。這封信在最上面。他沒有說話。我也沒問。

### 模板 E · 回望開場（適用於中後段章節）

```
如果那天我沒 [某個決定]，我今天會 [某個假設]。

但那天我 [具體發生的事]。
```

**範例：**
> 如果那天我沒去那個婚禮，我今天大概還會住在新竹，跟我媽同一條街上。
>
> 但那天我去了。我在婚禮上遇到她。她穿著綠色的洋裝，站在花園邊上。

## 深入展開的技巧

### 對話重現

當那個瞬間有關鍵對話，**逐字寫**：

```
[人物 A] 說：「...」

[使用者的反應 · 身體 · 沈默的時間]

[人物 A 又說]：「...」

[使用者說 / 沒說什麼]
```

**範例：**
> 她說：「我不是不愛你，我只是覺得我們不會走到最後。」
>
> 我沒有說話。我看著桌上的咖啡杯。杯緣有一點口紅。她剛剛喝過。
>
> 她說：「你可以說話。」
>
> 我說：「你什麼時候決定的？」
>
> 她說：「今天早上。」
>
> 我看她。「今天早上什麼時候？」
>
> 她說：「我醒來的時候。」

**注意：** 每一句對話後面都有一個**動作、身體、環境**的錨。不要純對話。

### 內心 OS

用**斜體**或**引號**或**段落內化**：

**引號版：**
> 「不要哭。」我對自己說。「至少現在不要。」

**內化版（Isaacson 常用）：**
> 我對自己說不要哭。至少現在不要。等她走了。等我走出這個咖啡廳。等我到停車場。到了停車場再哭。

**斜體版（如果 markdown 支援）：**
> *不要哭。至少現在不要。*

### 環境呼吸

每一段有情感重量的對話 / 事件後面，**加一段環境呼吸**：

**範例：**
> 她走了。
>
> 咖啡廳的音樂還在。是某首英文歌，我不知道歌名。角落有個媽媽在餵嬰兒喝奶。窗外開始下雨。
>
> 我坐在那裡大概 20 分鐘。侍者沒有來收杯子。我後來付了錢，走出去。

**這種呼吸段是「留白」的核心。**

## 光與影展現段（極重要）

每一章結尾前，要用**一段話**展現這章如何體現主題句的 tension。

**不要**用「這件事讓我意識到 XX」——那是說教。

**要**用**一個對照**、**一個回望**、**一個小 insight**：

**範例（主題句：「我一輩子在證明我不是我媽說的那個人」）：**
> 那天她走了之後，我開始拼命工作。三年之後我做到了副總。我媽在電話裡說：「阿彬，我以你為榮。」
>
> 我在辦公室聽到這句話。我笑了。
>
> 我後來想——我一輩子的方向，是因為那個女孩子那天說「我覺得我們不會走到最後」，還是因為我從 12 歲起就想聽到我媽說「以你為榮」？
>
> 或者，其實是同一件事。

**注意：** 這段話不下結論，用**問句**收，讓讀者感受。

## 留白結尾（5 種選一）

### 結尾 A · 畫面

```
[一個具體的、有畫面感的動作或場景]。
```

**範例：**
> 我開車回家。收音機沒開。經過復興北路的時候剛好綠燈。我踩油門，加速通過。

### 結尾 B · 一句話

```
[一句話，通常是內心 OS 或發現]。
```

**範例：**
> 那年我 27 歲。我以為我懂了什麼。

### 結尾 C · 對比

```
[之前]。[之後]。
```

**範例：**
> 25 歲那年，我覺得我是被留下的。
> 30 歲那年，我才知道，我是離開的那個。

### 結尾 D · 引用（謹慎用）

```
[使用者當時想的一句話 / 一句別人的話]。
```

**範例：**
> 我媽那天說：「你自己知道就好。」
>
> 那時候我不知道。現在也不完全知道。

### 結尾 E · 過場（連接下一章）

```
[一句話點出下一個瞬間的存在]。
```

**範例：**
> 我以為那是最後一次見到她。
> 但十年後，我在一個很奇怪的地方，又見到她。

## 特殊章節模板

### 序章模板

**目標：** 立刻讓讀者進入你的世界。600-1200 字。

```
[一個瞬間的深挖，可以是任何年紀]
[空間、感官、對話、內心 OS 都要有]
[結尾留一句話，暗示這本書的整個故事]
```

**範例結尾：**
> 這是我 65 歲那年的夏天。
>
> 但要說我這一生，我必須先回到 1965 年的鄉下。回到那條溪。回到那個在溪邊摸魚的小孩。

### 尾聲模板

**目標：** 回望，不總結，可能寫給某個人。500-1000 字。

**範例（寫給孫子）：**
> 小舟——
>
> 這本書寫的時候你 3 歲。你現在在客廳玩積木。你媽媽在做飯。
>
> 你長大之後會讀這本書。也許 15 歲，也許 30 歲。
>
> 我不知道你會從裡面看到什麼。
>
> 我只是希望，你在讀完之後，會走到窗邊，看看外面。然後想：**原來爺爺是這樣的一個人。**
>
> [引 T.S. Eliot]
> 「我們永遠不會停止探索。而所有探索的終點，是回到我們出發的地方——第一次真正認識它。」
>
> 我寫這本書，就是回到了起點。
>
> 現在，換你出發。
>
> 爺爺
> 2026 年 [月]

## 章節命名建議

好的章節名 = **一個物件 / 一句話 / 一個場景**，不是「第一章：我的童年」。

**好的範例：**
- 〈溪〉
- 〈那封信〉
- 〈老師的手〉
- 〈她說她不愛我〉
- 〈父親的最後一天〉
- 〈九月的下午〉

**不好的範例：**
- 「我的少年時代」
- 「事業的第一步」
- 「家庭的意義」

## 寫作前的檢查表

每一章開寫前，回頭看：

- [ ] 這章對應的**場景檔案**是哪份？（Phase 3 挖的）
- [ ] 這章的**主題句 tension** 是什麼？（Phase 4 定的）
- [ ] 這章的**開場方式**選哪一種？（A-E）
- [ ] 這章的**結尾方式**選哪一種？（A-E）
- [ ] 這章有**幾個關鍵對話**？逐字對話都有紀錄嗎？
- [ ] 這章有**幾個感官細節**？至少 3 個？

**每一項都 ✅ 之後才開始寫。**
